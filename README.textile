h1. Merbbit

Merbbit is a merb-based web app for community-driven link submission. It is heavily "inspired" by RubyFlow, Digg, Reddit and other such sites.

Merbbit is NOT production ready. I started this as a side-project to learn more about Merb.

h2. Features:

* Signup & login
* Link submission
* Link Voting
* Comments
* Threaded comments

h2. Planned Features:

* Comment voting
* Spam protection
* RSS feeds
* Search
* Proper specs...
